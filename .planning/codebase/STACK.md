# Tech Stack

## Core
- **Framework**: Next.js 16.2.3 (App Router)
- **Language**: TypeScript 5
- **Library**: React 19.2.4
- **Styling**: TailwindCSS 4.2.2, Framer Motion 12.38.0
- **State Management**: Zustand 5.0.12

## Database & Auth
- **Database**: MongoDB via Mongoose 9.4.1
- **Auth**: JWT (jose 6.2.2, jsonwebtoken 9.0.3), bcryptjs 3.0.3
- **Validation**: Zod 4.3.6

## Content & Media
- **Rich Text Editor**: Tiptap 3.22
- **Media Hosting**: Cloudinary 2.9
- **Icons**: Lucide React 1.8

## Infrastructure
- **Server**: Next.js (Node.js)
- **Validation**: Strict environment variable checks
- **Revalidation**: Dynamic cache invalidation (revalidateTag, revalidatePath)
